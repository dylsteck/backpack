<task type="auto">
  <name>Implement Config File Management</name>
  <files>packages/core/src/config/index.ts</files>
  <action>
Create config file read/write operations:

1. Determine config directory:
   - macOS: ~/Library/Application Support/cortex/
   - Linux: ~/.config/cortex/
   - Windows: ~/AppData/Roaming/cortex/

2. Config file path: <configDir>/config.json

3. Create getConfig():
   - Read config.json if exists
   - Merge with defaultConfig() for missing values
   - Validate with validateConfig()
   - Return typed Config object

4. Create setConfig(partialConfig):
   - Read existing config
   - Deep merge with new values
   - Validate
   - Ensure config directory exists
   - Write back to file (atomic write: write to temp, rename)
   - Set file permissions to 0600 (user read/write only)

5. Create resetConfig():
   - Delete config.json
   - Returns to defaults

6. Add helper functions:
   - getConfigPath(): returns full path
   - ensureConfigDir(): creates directory if needed

Handle errors gracefully with descriptive messages.
  </action>

<task type="auto">
  <name>Implement Keychain Integration</name>
  <files>packages/core/src/auth/keychain.ts</files>
  <action>
Create secure secret storage using OS keychain:

1. Add keytar as dependency (handles macOS Keychain, Windows Credential Manager, Linux libsecret)

2. Define SERVICE_NAME = 'cortex'

3. Create getSecret(key: string): Promise<string | null>:
   - Calls keytar.getPassword(SERVICE_NAME, key)
   - Returns secret or null if not found
   - Handle errors gracefully

4. Create setSecret(key: string, value: string): Promise<void>:
   - Calls keytar.setPassword(SERVICE_NAME, key, value)
   - Overwrites if exists
   - Log success (without value)

5. Create deleteSecret(key: string): Promise<void>:
   - Calls keytar.deletePassword(SERVICE_NAME, key)
   - Handle "not found" gracefully

6. Define well-known secret keys:
   - 'openrouter-api-key'
   - 'teller-access-token'
   - 'farcaster-signer'
   - 'farcaster-fid'

7. Create convenience functions:
   - getOpenRouterKey(): Promise<string | null>
   - setOpenRouterKey(key: string): Promise<void>
   - (similar for other secrets)

Add comprehensive error handling for missing keychain access.
  </action>
  <verify>node -e "const { setSecret, getSecret } = require('./packages/core/dist/auth/keychain'); await setSecret('test', 'value'); console.log(await getSecret('test'));"</verify>
