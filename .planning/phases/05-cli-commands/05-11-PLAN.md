Create timeline command:

1. Command structure:
   ```typescript
   export default class Timeline extends Command {
     static description = 'View timeline of all items';
     
     static flags = {
       limit: Flags.integer({ char: 'n', default: 50, description: 'Number of items' }),
       source: Flags.string({ char: 's', options: ['obsidian', 'farcaster', 'teller', 'chrome'], multiple: true }),
       since: Flags.string({ description: 'Show items since date (e.g., 1d, 2024-01-01)' }),
       until: Flags.string({ description: 'Show items until date' }),
       json: Flags.boolean({ description: 'Output as JSON' }),
     };
     
     async run() {
       const { flags } = await this.parse(Timeline);
       const db = getDatabase();
       
       // Build query
       let query = db.query.timelineItems.findMany({
         orderBy: (items, { desc }) => [desc(items.timestamp)],
         limit: flags.limit,
       });
       
       // Apply filters
       if (flags.source) {
         query = query.where(inArray(timelineItems.source, flags.source));
       }
       
       if (flags.since) {
         const sinceDate = parseDate(flags.since);
         query = query.where(gte(timelineItems.timestamp, sinceDate.getTime()));
       }
       
       // Execute query
       const items = await query;
       
       // Output
       if (flags.json) {
         this.log(JSON.stringify(items.map(formatItemJson), null, 2));
       } else {
         if (items.length === 0) {
           this.log('No items found.');
           return;
         }
         
         this.log(`Showing ${items.length} items:\n`);
         items.forEach(item => {
           this.log(formatItem(item));
           this.log(''); // Empty line between items
         });
       }
     }
   }
   ```

2. Date parsing:
   - parseDate(input: string): Date
   - Support: relative (1d, 1w, 1m) and absolute (2024-01-01)
   - Return Date object

3. Examples in help:
   - cortex timeline (last 50 items)
   - cortex timeline -n 100
   - cortex timeline -s obsidian
   - cortex timeline --since 1d
   - cortex timeline --json | jq

   ```typescript
   export default class Items extends Command {
     static description = 'List items with filtering';
     
     static flags = {
       source: Flags.string({ char: 's', options: ['obsidian', 'farcaster', 'teller', 'chrome'], required: true }),
       limit: Flags.integer({ char: 'n', default: 25 }),
       type: Flags.string({ description: 'Filter by type' }),
       search: Flags.string({ char: 'q', description: 'Search query' }),
       json: Flags.boolean({ description: 'Output as JSON' }),
       csv: Flags.boolean({ description: 'Output as CSV' }),
     };
     
     async run() {
       const { flags } = await this.parse(Items);
       const db = getDatabase();
       
       // Build query with filters
       const where = { source: flags.source };
       if (flags.type) where.type = flags.type;
       
       const items = await db.query.timelineItems.findMany({
         where,
         orderBy: (items, { desc }) => [desc(items.timestamp)],
         limit: flags.limit,
       });
       
       // Output formats
       if (flags.json) {
         this.log(JSON.stringify(items.map(formatItemJson), null, 2));
       } else if (flags.csv) {
         this.log('id,source,type,title,timestamp,url');
         items.forEach(item => {
           this.log(`${item.id},${item.source},${item.type},"${item.title || ''}",${item.timestamp},${item.url || ''}`);
         });
       } else {
         items.forEach(item => this.log(formatItem(item)));
       }
     }
   }
   ```

2. Source is required for items command
3. Support CSV export for data analysis
4. Examples:

   - cortex items -s farcaster -n 100 --json
   - cortex items -s teller --csv > transactions.csv
