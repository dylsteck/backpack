<task type="auto">
  <name>Implement Search Function</name>
  <files>packages/core/src/search/index.ts</files>
  <action>
Create main search function:

1. async search(options: SearchOptions): Promise<SearchResponse>

2. Implementation:
   - Start timer
   - Initialize results map: Map<string, SearchResult>
   
   - If useSemantic:
     * Call qmdClient.searchVectors(options.query, options.limit * 2)
     * For each vector result:
       - Fetch full item from DB
       - Add to results map with score and matchType='semantic'
   
   - If useFullText:
     * Build SQL query with FTS or LIKE:
       ```sql
       SELECT * FROM timeline_items 
       WHERE (title LIKE ? OR content LIKE ?)
       AND source IN (?) -- if filtered
       AND timestamp BETWEEN ? AND ? -- if date range
       LIMIT ?
       ```
     * For each text result:
       - If already in results map: update matchType='both', average scores
       - Else add to map with matchType='fulltext'
   
   - Apply filters:
     * Filter results by sources, date range
   
   - Sort by score (descending)
   - Apply limit and offset
   - Calculate highlights (extract matching text snippets)
   
   - Return SearchResponse with:
     * results: array
     * total: results.length
     * query: original query
     * durationMs: elapsed time

3. Helper functions:
   - calculateHighlight(item: TimelineItem, query: string): string[]
     * Extract sentences containing query terms
     * Return top 2-3 snippets
   - normalizeScore(score: number, type: 'semantic' | 'fulltext'): number
     * Normalize different score ranges to 0-1

4. Edge cases:
   - Empty query: return recent items
   - No results: return empty array
   - Database errors: throw with context

Export search function and types.
  </action>
