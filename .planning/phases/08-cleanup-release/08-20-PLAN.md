1. Update version to 2.0.0:
   - Update root package.json: "version": "2.0.0"
   - Update apps/cli/package.json: "version": "2.0.0"
   - Update apps/server/package.json: "version": "2.0.0"
   - Update packages/core/package.json: "version": "2.0.0"
   - Keep packages/api, auth, db, sdk as needed

2. Create/update CHANGELOG.md:
   ```markdown
   # Changelog

   ## [2.0.0] - 2024-XX-XX

   ### Major Changes
   - **Complete architecture rewrite**: Migrated from Electron desktop app to CLI-first design
   - **New CLI interface**: Rich terminal UI with Ink-based TUI mode
   - **Shared core package**: All business logic in @cortex/core
   - **SQLite everywhere**: Single database file, portable and fast
   - **OS keychain integration**: Secure secret storage
   - **tRPC API server**: Optional HTTP API for remote access
   - **Semantic search**: QMD integration for vector search
   - **Auto-embeddings**: Automatic embedding generation after sync

   ### New Commands
   - `cortex timeline` - View timeline with filters
   - `cortex items` - List items by source
   - `cortex search` - Semantic and full-text search
   - `cortex sync` - Sync data from all sources
   - `cortex view` - View item details
   - `cortex config` - Manage configuration
   - `cortex daemon` - Background sync daemon
   - `cortex auth` - Authenticate with services
   - `cortex tui` - Launch interactive TUI

   ### Removed
   - Electron desktop application
   - React-based UI
   - IPC communication layer
   - Window management
   - Complex build system

   ### Migration Guide
   1. Install new CLI: `npm install -g @cortex/cli`
   2. Configure sources: `cortex config --set sources.obsidian.vaultPath=/path`
   3. Authenticate: `cortex auth teller`, `cortex auth farcaster`
   4. Sync data: `cortex sync`
   5. Launch TUI: `cortex tui`

   ### Technical Details
   - Stack: OCLIF + Ink for CLI, Elysia + tRPC for server
   - Database: SQLite with better-sqlite3
   - Search: QMD for embeddings + SQLite FTS
   - Auth: OS keychain via keytar
   - Sync: Pluggable syncers for each source
   ```

3. Update apps/cli/package.json bin:

Prepare release.
