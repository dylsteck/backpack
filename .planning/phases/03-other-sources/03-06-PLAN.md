<task type="auto">
  <name>Implement Teller Syncer</name>
  <files>packages/core/src/sync/sources/teller.ts</files>
  <action>
Create TellerSyncer class:

1. Class structure:
   - name = 'teller'
   - config: TellerConfig
   - baseUrl = config.environment === 'sandbox' ? 'https://api.sandbox.teller.io' : 'https://api.teller.io'

2. isConfigured():
   - Check if access token exists in keychain
   - Return true/false

3. validateConfig():
   - Test API call to /accounts
   - Verify token is valid
   - Return true/false

4. doSync(progress):
   - Get access token from keychain
   - Fetch all accounts: GET /accounts
   - For each account:
     * Create/update account record
     * Fetch transactions: GET /accounts/{id}/transactions
     * Params: from date (last 90 days or last sync)
     * For each transaction:
       - Create TimelineItem:
         - id: uuid
         - source: 'teller'
         - type: 'transaction'
         - external_id: transaction.id
         - title: `${transaction.description} - $${transaction.amount}`
         - content: JSON.stringify(transaction)
         - raw_data: JSON.stringify({ account, transaction })
         - url: null
         - timestamp: new Date(transaction.date).getTime()
         - created_at: now
         - updated_at: now
       - Save via this.saveItem()
     * Update progress
   - Store last sync date
   - Return progress

5. API client methods:
   - async fetchAccounts(token: string): Promise<Account[]>
   - async fetchTransactions(token: string, accountId: string, fromDate?: string): Promise<Transaction[]>
   - API headers: Authorization: Basic base64(token:)

6. Error handling:
   - Handle 401 (invalid token)
   - Handle rate limits (429)
   - Handle account-specific errors

Export TellerSyncer.
  </action>
