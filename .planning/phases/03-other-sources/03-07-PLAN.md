<task type="auto">
  <name>Implement Chrome Syncer</name>
  <files>packages/core/src/sync/sources/chrome.ts</files>
  <action>
Create ChromeSyncer class:

1. Class structure:
   - name = 'chrome'
   - config: ChromeConfig

2. Default paths by browser:
   - Chrome macOS: ~/Library/Application Support/Google/Chrome/Default/History
   - Brave macOS: ~/Library/Application Support/BraveSoftware/Brave-Browser/Default/History
   - Chrome Linux: ~/.config/google-chrome/Default/History
   - Similar patterns for other browsers

3. isConfigured():
   - Get history file path
   - Check if file exists
   - Return true/false

4. validateConfig():
   - Try to open history database (read-only)
   - Verify can query urls table
   - Close connection
   - Return true/false

5. doSync(progress):
   - Get history file path
   - Copy to temp location (History file is locked while Chrome is running)
   - Open temp file with better-sqlite3 (read-only)
   - Calculate cutoff time:
     * Chrome timestamps are microseconds since Jan 1, 1601
     * Convert daysToSync to Chrome timestamp
   - Query:
     ```sql
     SELECT urls.id, urls.url, urls.title, urls.visit_count, 
            urls.last_visit_time, urls.typed_count
     FROM urls
     WHERE urls.last_visit_time > ?
       AND urls.url NOT LIKE 'chrome://%'
       AND urls.url NOT LIKE 'file://%'
       AND urls.url NOT LIKE 'devtools://%'
       AND urls.title IS NOT NULL
     ORDER BY urls.last_visit_time DESC
     ```
   - Deduplicate by URL (keep latest)
   - For each history item:
     * Convert Chrome timestamp to JS Date:
       - chromeTime = microseconds since 1601
       - unixTime = (chromeTime - 11644473600000000) / 1000
     * Create TimelineItem:
       - id: uuid
       - source: 'chrome'
       - type: 'visit'
       - external_id: url string (hashed or truncated)
       - title: page title
       - content: `Visited: ${url}`
       - raw_data: JSON.stringify(historyItem)
       - url: page URL
       - timestamp: unixTime
       - created_at: now
       - updated_at: now
     * Save via this.saveItem()
   - Clean up temp file
   - Update progress
   - Return progress

6. Helper methods:
   - getHistoryPath(): returns full path to History file
   - copyHistoryToTemp(sourcePath): returns temp path
   - chromeTimestampToUnix(chromeTime): returns milliseconds

7. Error handling:
   - Handle missing file
   - Handle locked database
   - Handle corrupt database

Export ChromeSyncer.
  </action>
